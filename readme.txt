				ReadMe for Contact List Application
Required Software – 
1.	Java
2.	MySQL
3.	Eclipse
4.	Tomcat
Instructions-
1.	Import the project into Eclipse
2.	Change the database config in user-beans.xml file in com.dbdesign.config folder
3.	Do a Maven Build using the command ‘clean install’
4.	Update the Maven project
5.	Run on Tomcat server
On server run, Home page with existing contacts will be displayed.
